### 天平与球

问:现有12个球与一个天平，已知12个球中有1个球与其它球不等重.有三次使用天平的机会，应如何找出该球?

答:记目标球为A.

i. 首先分别取4个球置于天平左右两端(记为a与b)，那么还剩4个球(记为c).
  + 若等重，则A在c中.
    ii. 取c中两球与a,b中任意两个已知等重球比较.
      + 若等重.则A在剩余两球中.iii.取其一与已知等重球比较即可.
      + 若不等中.则A在这两球中.iii.取其一与已知等重球比较即可.
  + 若不等重，则必有一高一低，设a高.
    ii. 取a,b中各两球置于天平左端(记为a',b')，再取a,b中各一球(记为a'',b'')与c中任意两个已知等重球置于右端比较.
      + 若等重.则A在剩余a,b各一球中.iii.取其一与已知等重球比较即可.
      + 若不等重.则需看左端高低.
        + 若左端高.说明A在a'两球或b''一球中.将a'两球分置天平两端.若平，b''为A.若不平，高者为A.
        + 若左端低.说明A在b'两球或a''一球中.将b'两球分置天平两端.若平，a''为A.若不平，低者为A.

先手六分进攻四分防守、后手六分防守四分进攻.

防守:士象稳定阵型、马炮最好在士象角上同时马守住要位炮进一步保住马，车出动占好要道.这样就在布局阶段形成了士象马炮车一体的防御体系，也为后续进攻做好准备.对方如果想要有效进攻必须兑子.进攻对方给对方增加压力(棋路的选择)，只有给对方增加压力才能增大对方犯错的可能.不能让对方进攻的太舒服.

进攻:压制对方大子活动范围(空间优势)，牵制对方主要兵力(使得己方子力充分发挥作用、对方子力无法发挥作用).寻找对方弱点、调动己方闲置兵力.稳固并确保优势(获得实质物质优势，过河卒)寻找进一步扩大的可能.寻找入局的手段.

基础杀法:铁门栓，重跑杀，天地炮，双杯献酒；钓鱼马，高钓马(侧面虎)，士角马(八角马)，卧槽马，双马饮泉，马后炮；双车错，夹车炮，大刀剜心，平顶冠，双鬼拍门.

基础残局:马擒单士，炮士胜双士，三兵胜士象全，双马/马炮胜士象全，士象全守和单车.

   + 中局:对各种价值的判断与选择.先手的价值，兑子是否会给对手遗留先手(先中后，后中先，先中先，后中后).一马约等于三四先.对方是否存在弱点，如何予以最有效的打击?对方防守的核心是什么，如何在这里做点文章?对方进攻的思路和路线是什么?

   + 混战攻杀与抢先入局:精确的计算力.

   + 残局:如何对对方子力进行最大程度的控制?

2. 采用具体手段实行决策.牵制、封锁、围困、兑子.

   + 提高行棋效率，一招多用:攻击对方子力的同时护住己方子力，攻击对方子力的同时稳固己方阵型/扩宽己方空间.贯彻行棋思路，中途改变思路就会降低效率.

## 洞箫

1. 节奏、气息起伏、装饰的处理是否符合这里的情绪?

   + 各个方面要素的对比可以增添音乐的内容.音乐内容的多少要符合主题和情绪，少了或多了都不合适.

   + 去掉不合适的叠音.

   + 气震幅度大往往就慢，幅度小往往频率也会加快.注意气震的对比.

   + 圆润度.不要有突兀的感觉.

   + 层次感.

2. 我想要达到这样的效果却吹不出来是因为哪方面的基本功不行?嘴唇控制.

   + 嘴唇的功夫对于要把箫吹好至关重要.曲子要富有层次感，就需要在强弱浓淡上实时做出变化.风门的大小，嘴唇的形态，口劲的大小在一句乐句中都会发生很明显地变化.嘴唇的功夫甚至可以说是仅次于气息的功夫了.一句乐句结束之后嘴唇应回复到初始状态.要想嘴唇在句中灵活变化在句终快速回复，嘴唇就不能死死顶住箫而是轻轻贴着.

   + 拉嘴角.

3. 单独练我能吹出这样的效果但在曲中却可能直接断掉?对各种状态的记忆(大脑记忆+肌肉记忆)不牢.

4. 对于吹箫来讲哪些是多余的?不合适的装饰音，多余的手指和肢体动作，不必要的心理和肌肉紧张.

## 防骗与反骗

人性之恶与人性之善都会永存.人性之恶也有诸多层次.装(糊涂),演(戏),恶心/诋毁,骗,偷,抢,暴力,群架,战争.大致从骗开始进入法律反对的领域.当然不同的恶有其自身特点不能相互取代.靠暴力无法得到的可能靠骗靠偷反而可行.

要提高自身防骗和反骗的能力，既包括身边熟人的骗/陌生人的骗，更包括诈骗.

+ 防骗与反骗.

  + 好的骗术都是七成的真+三成的假.

  + 熟人/陌生人的骗.往往临时起意或者出于习惯.

   + 有些人喜欢在日常生活中小骗一下人并以此为乐.

   + 长辈、老师、领导有时候会吓唬+骗+哄来达成一定的目的.

   + 相对陌生人的骗一般是因为某个活动某件事遇到一起，其为了事情的顺利进行在一定程度上进行骗(+捧，这往往是其的工作).

  + 诈骗与反诈骗.

   + 虽然身边相当多人都喜欢骗，但法律反对的只是以非法占有公私财物为目的的诈骗.诈骗是经过精心设计的.

   + 诈骗利用人性的若干弱点:贪婪(贪小便宜或者贪钱)、同情(且太容易相信别人)、好奇、恐惧(对公检法的恐惧)、自负(无知，觉得自己一定不会被骗).

   + 诈骗过程:目标筛选(青少年、刚毕业学生、老年人；处于人生低谷、急需用钱)，通过伪装骗取信任，实施骗取获取利益，洗钱消失转移完成诈骗周期.诈骗一般来讲总会识破的，所以诈骗的核心是在诈骗周期内获取的回报远远高于成本.相应的反诈骗的核心就是诈骗的代价远远高于回报，追诉周期远远长于诈骗周期.

   + 千门八将:以骗为职业.正(整体的布局)，提(负责宣传工作)，反(激将法诱人)，脱(发生意外时全身而退)，风(望风的)，火(暴力支持)，除(干脏活的善后工作)，谣(散布谣言把水搅浑).

   + 诈骗包括:街头诈骗，电信诈骗.

   + 电信诈骗:

     + 发源于台湾，发展于大陆，繁荣于东南亚.

     + 安溪:大陆的第一个据点；六合彩、刮刮乐；

   + 反诈骗:学习各种骗术的具体手段开阔眼界(火眼金睛)；重要事情找人商量()；涉及钱财延长付款周期，减少单次付款金额.
