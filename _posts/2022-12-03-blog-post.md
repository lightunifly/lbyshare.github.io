### 天平与球

问:现有12个球与一个天平，已知12个球中有1个球与其它球不等重.有三次使用天平的机会，应如何找出该球?

答:记目标球为A.

i. 首先分别取4个球置于天平左右两端(记为a与b)，那么还剩4个球(记为c).
  + 若等重，则A在c中.
    ii. 取c中两球与a,b中任意两个已知等重球比较.
      + 若等重.则A在剩余两球中.iii.取其一与已知等重球比较即可.
      + 若不等中.则A在这两球中.iii.取其一与已知等重球比较即可.
  + 若不等重，则必有一高一低，设a高.
    ii. 取a,b中各两球置于天平左端(记为a',b')，再取a,b中各一球(记为a'',b'')与c中任意两个已知等重球置于右端比较.
      + 若等重.则A在剩余a,b各一球中.iii.取其一与已知等重球比较即可.
      + 若不等重.则需看左端高低.
        + 若左端高.说明A在a'两球或b''一球中.将a'两球分置天平两端.若平，b''为A.若不平，高者为A.
        + 若左端低.说明A在b'两球或a''一球中.将b'两球分置天平两端.若平，a''为A.若不平，低者为A.

## 象棋

布局、中局与残局.

### 防守与进攻

布局

1. 中炮过河车进七兵体系.

   + 黑平炮兑车.

     + 红急进中兵.

       + 五步穿槽马.

     + 红左马盘河.

     + 红五六炮.

     + 红五九炮.

       + 红炮击中卒黑弃卒弃马变.

   + 黑左马盘河.

   + 黑高车保马.

先手六分进攻四分防守、后手六分防守四分进攻.

防守:士象稳定阵型、马炮最好在士象角上同时马守住要位炮进一步保住马，车出动占好要道.这样就在布局阶段形成了士象马炮车一体的防御体系，也为后续进攻做好准备.对方如果想要有效进攻必须兑子.进攻对方给对方增加压力(棋路的选择)，只有给对方增加压力才能增大对方犯错的可能.不能让对方进攻的太舒服.

进攻:压制对方大子活动范围(空间优势)，牵制对方主要兵力(使得己方子力充分发挥作用、对方子力无法发挥作用).寻找对方弱点、调动己方闲置兵力.稳固并确保优势(获得实质物质优势，过河卒)寻找进一步扩大的可能.寻找入局的手段.

基础杀法:铁门栓，重跑杀，天地炮，双杯献酒；钓鱼马，高钓马(侧面虎)，士角马(八角马)，卧槽马，双马饮泉，马后炮；双车错，夹车炮，大刀剜心，平顶冠，双鬼拍门.

基础残局:马擒单士，炮士胜双士，三兵胜士象全，双马/马炮胜士象全，士象全守和单车.

象棋的核心:对全局的掌控.

1. 对全局的审势与决策.是攻是守是等是谋和.要不停地问自己问题，在自问自答中找到自己.

   + 中局:对方是否存在弱点，如何予以最有效的打击?对方防守的核心是什么，如何在这里做点文章?对方进攻的思路和路线是什么?

   + 残局:如何对对方子力进行最大程度的控制?

2. 采用具体手段实行决策.牵制、封锁、围困、兑子.

   + 提高行棋效率，一招多用:攻击对方子力的同时护住己方子力，攻击对方子力的同时稳固己方阵型/扩宽己方空间.

   + 残局争兵卒.车扫兵最高效，炮清兵优于马，尤其是边兵，马吃边兵还要再跳一步到佳位.

   + 戒掉昏招.马跳河口被丝线牵牛.入险地而无直接攻击对象.

3. 深研常见特定局面.

   + 马炮残局.借炮运马.马控要道.

   + 使用叠炮叠车进攻.

   + 抗衡中炮盘头马急进中兵.

4. 基本功.

   + 子力的价值与功能.一个棋子在棋盘上所能起到的功能.

     + 作为一个子.

       + 蹩住马腿、塞住象眼.

       + 作为炮架.

     + 作为一个己方子或对方子.

       + 由于不能吃掉自己的棋子.在自己子力的行进路线上.挡住己方子力的通畅:挡住车路、炮路、兵无法往前拱、窝心马导致士无法相连、边马导致象无处可飞、中象挡住将往三楼的逃跑路线.

       + 由于可以吃掉对方的棋子.导致挡住路线的问题对于马兵士象将这五种短兵器(在一个方向只能走一步)并不存在.对于车炮部分存在.对于车只能行进到该棋子(吃掉该棋子)，被该棋子挡掉的就去不了了；对于炮只能行进到该棋子前一格(不能吃掉该棋子).当对于炮还有一个正面作用:作为炮的目标导致炮发生长距离转移.

       + 所以在影响棋路通畅的问题上.自己棋子造成的问题更大.

     + 作为一个兵种.

       + 车:威力最大.进攻防守的中坚力量.

       + 马:八面威风.进攻往往马到功成.防守则同时守住几条要道.

       + 炮:含威蓄势.具有很强的牵制作用.中炮一力往往攻势就成.

       + 兵:短兵器.数量多.用得好能当车使.在对方阵地连接马炮构建进攻体系.

## 防骗与反骗

人性之恶与人性之善都会永存.人性之恶也有诸多层次.装(糊涂),演(戏),恶心/诋毁,骗,偷,抢,暴力,群架,战争.大致从骗开始进入法律反对的领域.当然不同的恶有其自身特点不能相互取代.靠暴力无法得到的可能靠骗靠偷反而可行.

要提高自身防骗和反骗的能力，既包括身边熟人的骗/陌生人的骗，更包括诈骗.

+ 防骗与反骗.

  + 好的骗术都是七成的真+三成的假.

  + 熟人/陌生人的骗.往往临时起意或者出于习惯.

   + 有些人喜欢在日常生活中小骗一下人并以此为乐.

   + 长辈、老师、领导有时候会吓唬+骗+哄来达成一定的目的.

   + 相对陌生人的骗一般是因为某个活动某件事遇到一起，其为了事情的顺利进行在一定程度上进行骗(+捧，这往往是其的工作).

  + 反诈骗.

   + 虽然身边相当多人都喜欢骗，但法律反对的只是以非法占有公司财物为目的的诈骗.诈骗是经过精心设计的.

   + 诈骗包括:街头诈骗，电信诈骗.

   + 电信诈骗:目标筛选，通过伪装骗取信任，实施骗取获取利益，洗钱消失转移完成诈骗周期.诈骗一般来讲总会识破的，所以诈骗的核心是在诈骗周期内获取的回报远远高于成本.相应的反诈骗的核心就是诈骗的代价远远高于回报，追诉周期远远长于诈骗周期.
